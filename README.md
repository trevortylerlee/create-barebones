# create-barebones

Scaffold a new [Barebones](https://github.com/trevortylerlee/barebones) project.

## Usage

```bash
npm create barebones
```

Or with other package managers:

```bash
pnpm create barebones
yarn create barebones
bunx create-barebones
```

## What it does

1. Clones the Barebones template from GitHub
2. Prompts you for your site configuration (title, description, URL, etc.)
3. Writes your config into `siteConfig.ts` and `astro.config.mjs`
4. Optionally removes demo content
5. Installs dependencies
